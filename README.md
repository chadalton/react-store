# react-store
React store application using Mongoose/Mongo DB.

# API Features
Currently Supports the following end points:

1. Create a product: 
    - Method: POST
    - End Point: /product
    
1. Get a list of products: 
    - Method: GET 
    - End Point: /product
    
1. Create a wishlist: 
    - Method: POST
    - End Point: /wishlist
    
1. Get a list of wishlists: 
    - Method: GET
    - End Point: /wishlist
    
1. Add product to wishlist: 
    - Method: PUT
    - End Point: /wishlist
